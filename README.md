# 🏰 Tower Clash

A 2D top-down tower defense prototype built in **Unity** using **C#**, focused on modular gameplay systems, enemy AI, player progression, and scalable architecture.

The goal of this project was to design a maintainable tower defense framework while improving gameplay programming, object-oriented design, and system architecture.

---

## 🎮 Features

* 🏰 Wave-based enemy system
* ⚔️ Tower combat mechanics
* 👾 Enemy AI
* 📈 Player progression
* 💰 Currency & upgrade system
* 🎯 Health & damage system
* ⚡ Performance optimized
* 🎨 Responsive UI

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/DaLexusS/Tower-Clash.git
```

2. Open the project using the recommended Unity version.

3. Open the main scene.

4. Press **Play**.
